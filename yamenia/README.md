## 📋 نظرة عامة (Overview)

نظام **يمن سيف (YEMEN SAFE)** هو منصة ويب إدارية شاملة مصممة خصيصاً للخطوط الجوية اليمنية (YGHC)، تهدف إلى:

- **أتمتة تسجيل ومتابعة وتحليل الحوادث التشغيلية** بدءاً من لحظة التبليغ وحتى الإغلاق النهائي.
- **إدارة سجل المخاطر (Risk Registry)** وفقاً لمعايير السلامة التشغيلية (ICAO SMS).
- **تتبع مؤشرات الأداء (KPIs)** مثل متوسط وقت الاستجابة، نسبة إغلاق الحوادث، ودقة البيانات.
- **إدارة المستخدمين والصلاحيات** بنظام أدوار متعدد المستويات (RBAC).
- **دعم ثنائي اللغة (عربي / إنجليزي)** مع تصميم متجاوب يعمل على جميع الأجهزة.

## 🛠️ التقنيات المستخدمة (Tech Stack)

### الواجهة الأمامية (Frontend)
| التقنية | الاستخدام |
|---------|-----------|
| **HTML5** | الهيكل الأساسي مع دعم RTL و Semantic HTML |

### الخلفية (Backend)
| التقنية | الاستخدام |
|---------|-----------|
| **PHP 7.4+** | RESTful API مع PDO Prepared Statements |
| **MySQL / MariaDB** | قاعدة بيانات علائقية (InnoDB) مع utf8mb4 |
| **bcrypt** | تشفير كلمات المرور (`password_hash`) |
| **Sessions** | إدارة جلسات المصادقة |

### الوضع الحالي
> ⚙️ **الواجهة الأمامية تعمل حالياً ببيانات تجريبية (MockData)** محفوظة في `js/data.js` مما يسمح بتشغيلها مباشرة بدون خادم. الباك إند جاهز ويمكن ربطه بقاعدة بيانات MySQL لتشغيل كامل.

## 📖 وصف الصفحات (Pages Description)

### 1. 🔐 صفحة تسجيل الدخول (`index.html`)
- خلفية متحركة مع رسوم طائرات
- التحقق من بيانات المستخدم
- التوجيه التلقائي حسب صلاحيات الدور
- إعادة التوجيه للمسجلين مسبقاً

### 2. 📊 لوحة التحكم (`dashboard.html`)
- بانر ترحيبي مخصص باسم المستخدم + ساعة حية
- 4 بطاقات إحصائية (حوادث اليوم/الأسبوع/الشهر + معدل الإغلاق)
- 4 بطاقات KPI (وقت الاستجابة، نسبة الإغلاق، دقة البيانات، رضا المستخدمين)
- رسم بياني شريطي للحوادث الشهرية (سنوي / ربع سنوي)
- رسم بياني دائري لتوزيع أنواع الحوادث
- مؤشرات سجل المخاطر (غير مقبول، قيد التنفيذ، تم حلها، نسبة التخفيف)
- جدول آخر الحوادث + سجل آخر النشاطات

### 3. ⚠️ إدارة الحوادث (`incidents.html`)
- جدول شامل بجميع الحوادث مع تصفية (نوع، حالة، أولوية)
- نموذج إضافة حادث جديد مع رفع صور
- تعديل وحذف الحوادث (حسب الصلاحية)
- أيقونات وألوان حسب النوع والأولوية

### 4. 📋 تفاصيل الحادث (`incident-detail.html`)
- عرض كامل لبيانات الحادث
- تتبع الإجراءات المتخذة (Timeline)
- إمكانية تغيير الحالة والأولوية
- عرض الصور المرفقة

### 5. 🛡️ سجل المخاطر (`risks.html`)
- عرض المخاطر بنظام البطاقات والجدول
- مصفوفة المخاطر (Probability × Severity)
- تصنيف ثلاثي: مقبول / مقبول مع مراجعة / غير مقبول
- بيانات مستخرجة من YGHC Risk Registry الفعلي (أبريل 2025)

### 6. 📈 التقارير (`reports.html`)
- فلاتر حسب الفترة الزمنية
- رسوم بيانية إحصائية
- تصدير CSV + PDF + طباعة (لمدير العمليات فقط)

### 7. 🔔 الإشعارات (`notifications.html`)
- عرض الإشعارات مع تصنيف (حوادث، تحديثات، مهام، نظام)
- تعليم كمقروء (فردي / جماعي)

### 8. ⚙️ الإعدادات (`settings.html`)
- الملف الشخصي (الاسم، البريد، الهاتف)
- تغيير كلمة المرور
- إعدادات الإشعارات
- إعدادات النظام (اللغة، المنطقة الزمنية)
- معلومات النظام (الإصدار، الترخيص)
- إدارة المستخدمين (للمسؤولين فقط)

## 🔧 الميزات التقنية (Technical Features)

### الواجهة الأمامية
- ✅ **CSS Variables** — نظام متغيرات شامل (ألوان، خطوط، مسافات، ظلال، أنصاف أقطار)
- ✅ **Dark Mode** — وضع داكن كامل مع حفظ التفضيل في `localStorage`
- ✅ **i18n (DOM Walker)** — ترجمة حية عربي/إنجليزي مع `MutationObserver` للمحتوى الديناميكي
- ✅ **Responsive Design** — تصميم متجاوب مع نقاط توقف متعددة (Mobile, Tablet, Desktop)
- ✅ **Animated Numbers** — تحريك أرقام الإحصائيات عند التحميل
- ✅ **Custom Tooltips** — تلميحات مخصصة بدلاً من الافتراضية
- ✅ **Toast Notifications** — إشعارات منبثقة مع شريط تقدم وأيقونات
- ✅ **Skeleton Loading** — هياكل تحميل مختلفة (Cards, Tables, Charts, Spinner)
- ✅ **LocalStorage Persistence** — حفظ واسترجاع البيانات المعدلة محلياً
